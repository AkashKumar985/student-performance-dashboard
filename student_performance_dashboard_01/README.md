# Student Performance Dashboard

## Description
An interactive dashboard to visualize student performance across subjects.

## Features
- Login system
- Visual charts for subject-wise and student-wise performance
- Easy filtering and insights

## Tech Stack
- Python, Flask
- HTML, CSS, JavaScript (Chart.js)
- Pandas for data processing

## Setup
```bash
pip install -r requirements.txt
python src/app.py
```
